Guidelines for using SoneMC plugins and website content

General

What You Can Do

  • Use our plugins on your own Minecraft server
  • Adjust plugin configuration files as documented
  • Share feedback and ideas to improve features
  • Create tutorials, reviews, or videos (with credits)

What You Can’t Do

  • Do not use
  • No reselling — even if modified
  • No code edits — modifying source files is not allowed
