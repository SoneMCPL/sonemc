Valentine

Developed: 1.20.2
Tested: 1.16, 1.17, 1.18.2, 1.19, 1.20.2, 1.21

All those hearts in that one day. Hugs & kisses, marriages etc..

Features:

  • Changes custom words with custom symbol and color (in chat)
  • You can send a Kiss/Hug to a player (with cooldown)
  • Heart particles around player
  • Hugs/Kisses leaderboard
  • Multilanguage support
  • Marriage system
  • Permissions

Commands:

Command Alias Description Permission
/valentines /val, /vday Displays plugin's commands valentine.use
/valentinesplayer (player) /vstats (player) Displays player's stats valentine.use
/valentinesleaderboard /vlb Displays leaderboard valentine.use
/hug (player) - Hugs player valentine.use
/kiss (player) - Kisses player valentine.use
/marry (player) - Marry another player valentine.marry
/divorce - Divorce your partner valentine.divorce
/couples - Display all married couples valentines.use
/valtest - Plugin testing command CONSOLE

Permissions:

Permission Description
valentines.use Allows the player to use basic Valentines commands
valentines.effect Allows the player to see particle effects around them
valentines.marry Allows the player to marry other players
valentines.divorce Allows the player to divorce their partner
valentines.admin Allows access to administrative commands
CONSOLE Command works only in console

Harvested - 1.0

Developed: 1.20.2
Tested: 1.20, 1.21

Placing seeds and breaking them by radius

Features:

  • Shift + Right-Click with seeds in hand to automatically plant in a 5x5 area around you (configurable radius)
  • Change the planting range in config
  • Breaking a mature crop (like potatoes)? All nearby grown crops of the same type will break too
  • Harvested crops are instantly added to your inventory

config.yml:

Copied!
radius: 5
direct-inventory: true
