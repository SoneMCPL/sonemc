Installation

Many people know it but, just to ensure you know it too:

  1. Download the plugin JAR file from our website or SpigotMC.
  2. Place the JAR file in your server's plugins folder.
  3. Start/Restart your server.
  4. Enjoy.

Troubleshooting

If you encounter any issues while using our plugins, check the following common solutions:

Plugin Doesn't Load

  1. Check if you're using a compatible server version.
  2. Verify that the plugin JAR file is in the correct folder.
  3. Look for error messages in the server console or logs.
  4. Try to reinstall plugin.

Configuration Issues

  1. Try reverting to the default configuration and making changes gradually.
  2. Check the plugin's documentation for specific configuration requirements.
