MobStatus

Version: 1.0-Experimental
Status: Experimental
Tested on: 1.20.x - 1.21.x
Last updated: May 3, 2025

Advanced RPG-style health bars, damage indicators, and effects for mobs

Download Latest View on GitHub View on SpigotMC

Features

  • Effects - Shows particles for critical hits, normal damage, enchanted golden apple effects, and more
  • HealthBar - Displays mob health above their heads using 3 style types (standard, gradient, segmented)
  • GUI - Displays informations about killed mobs
  • Tiers - Every mob is categorized from Common to Legendary based on strength and rarity
  • More - Change loots, effects, tiers, and more via config.yml

Commands

MobStatus uses only 3 commands.

All commands used in plugin:

Command Alias Description Permission
/mobstatus status - Opens GUI mobstatus.stats
/mobstatus reload - Reloads plugin mobstatus.admin
/mobstatus help - Displays all commands -

Permissions

This plugin uses few permissions to control acces to commands and features:

Permission Groups:

  • mobstatus.admin - Allows to use all commands
  • mobstatus.stats - Allows opening GUI

Configuration

A lot of things in this plugin are customizable in config.yml file.

Main Configuration (config.yml)

config.yml 
# World Configuration
 worlds:
 # List of worlds where the plugin is enabled
 # Use * to enable in all worlds
 enabled-worlds:
 - "*"
 
 # Health Bar Configuration
 health-bar:
 # Character to use for health bar
 character: "■"
 
 # Characters for empty health bar
 empty-character: "■"
 
 # Health bar style: "standard", "gradient", or "segmented"
 style: "gradient"
 
 # Health bar length (number of characters)
 length: 10
 
 # Show borders around health bar
 show-borders: true
 
 # Border characters
 prefix: "&8["
 suffix: "&8]"
 
 # Show entity info line (health percentage, name, etc.)
 show-info: true
 
 # Show entity level
 show-level: true
 
 # Colors for different health percentages
 colors:
 100-81: "&2" # Green (Full/High health)
 80-61: "&a" # Light Green (High health)
 60-51: "&e" # Yellow (Medium health)
 50-41: "&6" # Gold (Medium health)
 40-31: "&c" # Red (Medium-low health)
 30-21: "&c" # Red (Low health)
 20-11: "&4" # Dark Red (Low health)
 10-1: "&4" # Dark Red (Very low health)
 0: "&0" # Black (Dead)
 
 # Color for empty health bar
 empty-color: "&8" # Dark Gray
 
 # Damage Indicators
 damage-indicators:
 # Enable floating damage numbers and types
 enabled: true
 
 # Duration in ticks (20 ticks = 1 second)
 duration: 60
 
 # Effect Particles Configuration
 effects:
 # Default duration for effects
 default-duration: 30
 
 # Healing effect

 enabled: true
 particle-count: 8
 duration: 30
 
 # Critical hit effect

 enabled: true

 enabled: true

 duration: 20
 
 # Poison effect

 # Fire effect

 # Status effects (slow, etc.)

 enabled: true

 # Death effects

 enabled: true

 # Display Options

 # Duration of effect particles in ticks (20 ticks = 1 second)

 # Update frequency for health bars in ticks

 # Distance at which health bars become visible
 visibility-range: 24
 
 # Offset above mob's head
