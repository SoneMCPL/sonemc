Harvesting

Version: 1.0
Status: Stable
Tested on: 1.20.x - 1.21.x
Last updated: April 20, 2025

Survival plugin for better farming experience. This plugin let you place seeds faster by shifting and auto-placing/cropping around you.

Download Latest View on GitHub View on SpigotMC

Features

  • Autoplace - Shift+Right-click on farmland with seed, and plugin will autoplace seeds around player (using Radius)
  • Customizable - Radius is customizable
  • Magnet - Breaking fully grown crops will get into inventory
  • Filter - Breaking only one type of crop.

Configuration

Every part of Harvesting plugin is customizable in config.yml file.

Main Configuration (config.yml)

config.yml 
radius: 5
direct-inventory: true
