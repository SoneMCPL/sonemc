LetsPlay

Version: 1.0-Experimental-2
Status: Experimental
Tested on: 1.19.x - 1.21.x
Last updated: May 8, 2025

Simple lobby system designed for smaller server/communities

Download Latest View on SpigotMC View on Modrinth

Features

  • Compass-based Server Selector (SS) - On the lobby world, players get a compass to select mini-game servers.
  • Custom commands via Server Selector - Configurable in config.yml
  • Lobby settings - Block damage, interaction, and more.
  • Command aliases - /lobby,/hub,/spawn and /lpreload
  • Time played GUI - Shows playerâ€™s profile with playtime (In SS GUI)

Commands

LetsPlay provides few commands to help it make it more controlable:

All commands used in plugin:

Command Alias Description Permission
/setlobby - Set the current location as the lobby letsplay.setlobby
/lobby /hub, /spawn Teleport to the lobby letsplay.lobby
/lobbysettings - Open GUI to configure lobby behavior letsplay.lobbysettings
/letsplayreload /lpreload Reload the plugin configs letsplay.reload

Permissions

This plugin uses few permissions to control acces to commands and features:

Permission Groups

  • letsplay.* - Access to everything
  • letsplay.setlobby - Set the lobby
  • letsplay.lobby - Teleport to lobby
  • letsplay.lobbysettings - Open lobby settings GUI
  • letsplay.reload - Reload configs

Bypass

  • letsplay.bypass.*
  • letsplay.bypass.blockbreak
  • letsplay.bypass.blockplace
  • letsplay.bypass.containers

Every part of LetsPlay plugin is customizable in config files.

# Settings
settings:
  # Lobby protection settings
  lobby-protection:
    enabled: true
    prevent-fall-damage: true
    prevent-pvp: true
    prevent-all-damage: true
    prevent-block-break: true
    prevent-block-place: true
    prevent-hunger: true
    prevent-drowning: true
    prevent-container-access: true
    prevent-interaction: true
    force-adventure-mode: true
    teleport-on-join: true
    clear-inventory: true

  # Items settings
  items:
    compass:
      enabled: true
      slot: 0 # 0 = first slot
      name: "&bServer Selector"
      lore:
        - "&7Right-click to open"
        - "&7the server selector!"

    info-book:
      enabled: true
      slot: 8 # 8 = Last slot
      name: "&aServer Information"
      lore:
        - "&7Right-click to view"
        - "&7server information!"
